# 테스트하기

만든 VR 씬을 테스트해봅시다! Quest 3가 있으면 VR로, 없으면 시뮬레이터로 테스트할 수 있어요.

---

## Quest Link로 테스트하기 (VR 헤드셋 있음)

Quest 3를 PC와 연결해서 실제 VR로 테스트해요.

### USB 케이블 연결 (Quest Link)

**준비물:** USB 3.0 케이블

**연결 순서:**

1. **PC에서** Meta Quest 앱이 실행 중인지 확인
   - 작업 표시줄에서 Meta Quest 아이콘 찾기
   - 없으면 시작 메뉴에서 "Meta Quest" 검색해서 실행

2. **Quest 3**를 USB-C 케이블로 **PC에 연결**

3. Quest 3를 쓰면 팝업이 나와요:
   ```
   "데이터 액세스를 허용하시겠습니까?"
   ```
   → **"허용"** 선택

4. Quest 3의 **Quick Settings** 열기
   - 왼쪽 컨트롤러의 메뉴 버튼 누르기
   - 또는 시계 영역 클릭

5. **Quest Link** 선택

6. 연결된 **PC 이름** 선택 후 **실행**

### 무선 연결 (Air Link)

케이블 없이 Wi-Fi로 연결하는 방법이에요.

**필수 조건:**
- PC와 Quest 3가 **같은 Wi-Fi**에 연결
- **5GHz Wi-Fi** 권장 (2.4GHz는 느려요)
- 공유기가 같은 방에 있으면 좋아요

**연결 순서:**

1. Quest 3에서 **Quick Settings** 열기
2. **Quest Link** 선택
3. **"Air Link 사용"** 켜기
4. PC 목록에서 **PC 선택**
5. 처음이면 **페어링 코드** 입력
6. **실행**

---

### Unity에서 VR 테스트하기

Quest Link가 연결된 상태에서:

1. Unity 열기
2. 테스트할 씬 열기
3. **Play 버튼** 클릭 (또는 `Ctrl + P`)
4. Quest 3 화면에서 VR 씬이 보여요!

**테스트해볼 것들:**

- [ ] 머리를 돌리면 화면도 돌아가는지
- [ ] 컨트롤러가 보이는지
- [ ] 물체를 잡을 수 있는지
- [ ] 바닥을 텔레포트할 수 있는지

**테스트 종료:**

Unity에서 다시 **Play 버튼** 클릭 (또는 `Ctrl + P`)

## VR 헤드셋 없이 테스트하기 (시뮬레이터)

Quest 3가 없어도 키보드와 마우스로 VR을 테스트할 수 있어요!

### XR Device Simulator 설정하기

**1단계: 시뮬레이터 프리팹 찾기**

1. Project 창에서 **검색창**에 `XR Device Simulator` 입력
2. 또는 직접 찾기:
   ```
   Assets/Samples/XR Interaction Toolkit/[버전]/XR Device Simulator/
   ```

**2단계: 씬에 추가하기**

1. **XR Device Simulator** 프리팹을 찾았으면
2. Hierarchy 창으로 **드래그 앤 드롭**
3. 끝!

**3단계: 테스트하기**

1. **Play 버튼** 클릭
2. Game 창에서 시뮬레이터 조작

---

### 시뮬레이터 조작법

| 동작 | 키/마우스 |
|------|-----------|
| **머리 돌리기** | 마우스 이동 |
| **이동하기** | W / A / S / D |
| **위로 올라가기** | E |
| **아래로 내려가기** | Q |
| **왼손 조작** | 왼쪽 Shift 누른 상태로 |
| **오른손 조작** | Space 누른 상태로 |
| **트리거 (잡기)** | 마우스 왼쪽 클릭 |
| **그립** | G |

### 예시: 물체 잡기

1. Play 모드 시작
2. `Space` 누른 상태로 (오른손 조작 모드)
3. 마우스로 물체 쪽을 바라보기
4. `G` 또는 마우스 클릭으로 잡기

### 예시: 텔레포트하기

1. Play 모드 시작
2. `Space` 누른 상태로
3. 바닥을 바라보기
4. 마우스 클릭 → 텔레포트!

## 테스트 중 문제가 있을 때

### VR 화면이 안 보여요

1. Quest Link가 제대로 연결됐는지 확인
2. Meta Quest 앱이 PC에서 실행 중인지 확인
3. Unity Project Settings에서 OpenXR이 체크됐는지 확인

### 컨트롤러가 안 보여요

1. OpenXR 설정에서 "Oculus Touch Controller Profile" 추가됐는지 확인
2. Quest 3 컨트롤러 배터리 확인

### 시뮬레이터가 안 돼요

1. XR Device Simulator가 씬에 있는지 확인
2. Starter Assets 샘플이 설치됐는지 확인
3. Play 모드에서 Game 창을 클릭해서 포커스 주기

### 화면이 끊겨요 (렉)

1. USB 3.0 케이블 사용 중인지 확인 (USB 2.0은 느려요)
2. 다른 프로그램 종료하기
3. Air Link 쓰면 유선으로 바꿔보기

## 테스트 팁

### 자주 저장하기

VR 테스트 전에 항상 `Ctrl + S`로 저장하세요.
Unity가 가끔 멈출 수 있어요.

### Console 창 확인하기

Play 모드에서 문제가 있으면 Console 창을 확인하세요.
빨간색 에러 메시지가 문제의 원인을 알려줘요.

### 작은 것부터 테스트하기

복잡한 기능을 한번에 테스트하지 말고,
하나씩 추가하면서 테스트하세요.
