# 첫 번째 VR 씬 만들기

드디어 VR 세계를 만들어볼 시간이에요!

---

## 방법 1: 에디터 도구 사용하기 (쉬운 방법)

이 프로젝트에는 자동 설정 도구가 있어요.

### 도구 사용법

1. Unity 상단 메뉴: **VR Game** > **프로젝트 설정 도우미**
2. 열리는 창의 안내를 따라하면 끝!

> 메뉴가 없다면 아래 "직접 만들기"를 따라하세요.

---

## 방법 2: 직접 만들기

하나씩 직접 만들어보면 이해하기 좋아요!

### Step 1: 새 씬 만들기

**씬(Scene)** 은 게임의 한 장면이에요. 메뉴 화면, 게임 화면, 엔딩 화면이 각각 씬이 될 수 있어요.

1. Unity 상단 메뉴: **File** > **New Scene**
2. **"Basic (Built-in)"** 선택하고 **"Create"** 클릭
3. 빈 씬이 만들어져요!

### 씬 저장하기

1. **Ctrl + S** (Mac: Cmd + S) 누르기
2. 저장 위치: `Assets/Scenes/` 폴더
3. 파일 이름: `MainScene` (원하는 이름으로)
4. **"Save"** 클릭

---

### Step 2: 기본 카메라 삭제하기

VR에서는 일반 카메라 대신 VR 전용 카메라를 써야 해요.

1. Hierarchy 창에서 **"Main Camera"** 찾기
2. 오른쪽 클릭 > **"Delete"** 선택

> 또는 Main Camera 선택 후 Delete 키 누르기

---

### Step 3: XR Origin 추가하기 (VR 플레이어)

XR Origin은 VR에서 플레이어의 위치와 카메라, 손을 담당해요.

1. Unity 상단 메뉴: **GameObject** > **XR** > **XR Origin (VR)**
2. 클릭하면 Hierarchy에 XR Origin이 생겨요!

### XR Origin 구조 알아보기

```
XR Origin
├── Camera Offset
│   ├── Main Camera        ← VR 카메라 (머리)
│   ├── Left Controller    ← 왼손 컨트롤러
│   └── Right Controller   ← 오른손 컨트롤러
```

- **Main Camera**: 플레이어가 보는 화면
- **Left/Right Controller**: 양손 컨트롤러

---

### Step 4: 바닥 만들기

VR에서 서 있을 바닥을 만들어요.

**바닥 오브젝트 만들기:**

1. **GameObject** > **3D Object** > **Plane** 클릭
2. Hierarchy에 **Plane**이 생겨요

**위치 조정하기:**

1. Hierarchy에서 **Plane** 선택
2. Inspector 창에서 **Transform** 확인
3. 다음과 같이 설정:
   - Position: X=`0`, Y=`0`, Z=`0`
   - Rotation: X=`0`, Y=`0`, Z=`0`
   - Scale: X=`1`, Y=`1`, Z=`1`

---

### Step 5: 텔레포트 기능 추가하기 (선택)

VR에서 먼 곳으로 순간이동하는 기능이에요. 걷는 것보다 멀미가 덜해요!

**바닥에 텔레포트 추가:**

1. Hierarchy에서 **Plane** 선택
2. Inspector 창 맨 아래 **"Add Component"** 클릭
3. 검색창에 `Teleportation Area` 입력
4. **Teleportation Area** 클릭해서 추가

이제 바닥 아무 곳이나 가리키고 텔레포트할 수 있어요!

---

### Step 6: 잡을 수 있는 물체 만들기

VR의 재미는 물건을 잡는 거죠!

**큐브 만들기:**

1. **GameObject** > **3D Object** > **Cube** 클릭
2. Hierarchy에 **Cube**가 생겨요

**위치와 크기 조정:**

Inspector의 Transform에서:
- Position: X=`0`, Y=`1`, Z=`0.5` (플레이어 앞, 허리 높이)
- Scale: X=`0.1`, Y=`0.1`, Z=`0.1` (손으로 잡기 좋은 크기)

**물리 법칙 추가하기:**

1. Cube 선택된 상태에서
2. **Add Component** > `Rigidbody` 검색 > 추가
3. 이제 큐브가 중력의 영향을 받아요

**잡기 기능 추가하기:**

1. 같은 방법으로 **Add Component** 클릭
2. `XR Grab Interactable` 검색 > 추가
3. 이제 손으로 잡을 수 있어요!

---

## 테스트해보기

씬을 만들었으니 테스트해볼까요?

### VR로 테스트 (Quest 3 연결된 경우)

1. Quest 3가 Quest Link로 PC에 연결되어 있는지 확인
2. Unity에서 **Ctrl + P** (Play 버튼) 누르기
3. Quest 3 화면에서 VR 씬을 볼 수 있어요!

### 시뮬레이터로 테스트 (Quest 없는 경우)

VR 헤드셋이 없어도 테스트할 수 있어요.
[테스트하기 문서](06-테스트하기.md)에서 자세히 알아보세요.

---

## 씬 구성 확인하기

제대로 만들었는지 확인해볼까요?

### Hierarchy 창이 이렇게 보여야 해요:

```
MainScene
├── XR Origin
├── Plane (바닥)
├── Cube (잡을 물체)
└── Directional Light (조명)
```

### 각 오브젝트 컴포넌트 확인:

| 오브젝트 | 필요한 컴포넌트 |
|----------|-----------------|
| Plane | Mesh Renderer, Mesh Collider, **Teleportation Area** |
| Cube | Mesh Renderer, Box Collider, **Rigidbody**, **XR Grab Interactable** |

## 자주 묻는 질문

### Q: XR Origin 메뉴가 없어요
XR Interaction Toolkit이 설치 안 됐어요.
[Quest 3 연결 설정](04-Quest3-연결.md)의 Step 5를 확인하세요.

### Q: 큐브가 바닥을 뚫고 떨어져요
Plane에 Collider가 없을 수 있어요.
Plane 선택 > Add Component > Mesh Collider 추가

### Q: 잡아도 손에 안 붙어요
XR Grab Interactable이 제대로 추가됐는지 확인하세요.
Cube 선택 > Inspector에서 컴포넌트 목록 확인
