# 문제 해결

자주 발생하는 문제와 해결 방법을 모았어요!

---

## Quest Link 연결 문제

### Quest Link가 연결되지 않아요

**확인할 것들:**

1. **USB 케이블 확인**
   - USB 3.0 이상 케이블인지 확인
   - USB 2.0 케이블은 느리거나 안 될 수 있어요
   - 케이블이 손상되지 않았는지 확인

2. **PC의 Meta Quest 앱 확인**
   - 앱이 실행 중인지 확인
   - 로그인되어 있는지 확인
   - Quest와 **같은 계정**으로 로그인했는지 확인

3. **USB 포트 바꿔보기**
   - 다른 USB 3.0 포트에 연결해보세요
   - USB 허브 말고 PC에 직접 연결하세요

4. **Quest 3 재시작**
   - Quest 3 전원을 껐다가 다시 켜세요
   - 전원 버튼 길게 눌러서 "다시 시작" 선택

5. **PC 재시작**
   - 가끔 PC 재시작으로 해결돼요

### Air Link가 안 돼요

1. **같은 Wi-Fi인지 확인**
   - PC와 Quest 3가 같은 Wi-Fi에 연결되어 있어야 해요

2. **5GHz Wi-Fi 사용**
   - 2.4GHz는 느려서 끊길 수 있어요
   - 공유기 설정에서 5GHz 네트워크에 연결하세요

3. **공유기와 가까이**
   - 공유기가 같은 방에 있으면 좋아요
   - 벽을 많이 지나면 신호가 약해져요

## Unity 에러

### VR이 Unity에서 안 보여요

**해결 방법:**

1. Quest Link 연결 상태 확인
2. **Edit > Project Settings > XR Plug-in Management** 열기
3. **OpenXR** 체크되어 있는지 확인
4. **OpenXR** 설정에서 **Oculus Touch Controller Profile** 추가됐는지 확인
5. Unity 재시작

### XR Origin 메뉴가 없어요

XR Interaction Toolkit이 설치 안 됐어요.

**설치 방법:**

1. **Window > Package Manager** 열기
2. 왼쪽 위 **+** 버튼 클릭
3. **"Add package by name"** 선택
4. 입력: `com.unity.xr.interaction.toolkit`
5. **"Add"** 클릭
6. 설치 완료 기다리기

### 빨간색 에러가 잔뜩 나와요

**시도해볼 것들:**

1. **잠시 기다리기**
   - 처음 열 때 패키지 설치 중일 수 있어요
   - 아래 진행 바가 끝날 때까지 기다리세요

2. **Unity 재시작**
   - Unity를 껐다가 다시 켜보세요

3. **패키지 다시 설치**
   - Package Manager에서 문제 패키지 제거 후 다시 설치

4. **Library 폴더 삭제**
   - Unity 종료
   - 프로젝트 폴더의 `Library` 폴더 삭제
   - Unity 다시 열기 (오래 걸려요)

## 컨트롤러 문제

### 컨트롤러가 인식되지 않아요

1. **배터리 확인**
   - 컨트롤러 배터리가 있는지 확인
   - 충전하거나 새 배터리로 교체

2. **페어링 확인**
   - Quest 3 설정에서 컨트롤러가 연결되어 있는지 확인
   - 연결 안 됐으면 다시 페어링

3. **OpenXR 설정 확인**
   - Project Settings > XR Plug-in Management > OpenXR
   - **Oculus Touch Controller Profile** 추가됐는지 확인

### 컨트롤러 위치가 이상해요

1. **Guardian 재설정**
   - Quest 3에서 Guardian(경계) 다시 설정

2. **조명 확인**
   - 너무 어두우면 추적이 안 돼요
   - 방을 밝게 하세요

## 물체 상호작용 문제

### 물체를 잡을 수 없어요

**확인할 것들:**

1. **Collider 확인**
   - 물체에 Collider(Box, Sphere, Mesh 등)가 있어야 해요
   - Inspector에서 확인

2. **XR Grab Interactable 확인**
   - 물체에 `XR Grab Interactable` 또는 `GrabbableObject` 컴포넌트가 있어야 해요

3. **컨트롤러에 Interactor 확인**
   - XR Origin > Camera Offset > Left/Right Controller
   - 각 컨트롤러에 `XR Direct Interactor` 있어야 해요

### 물체가 바닥을 뚫고 떨어져요

**해결 방법:**

1. 바닥(Plane)에 **Collider** 있는지 확인
2. 없으면 **Add Component > Mesh Collider** 추가

## 핸드 트래킹 문제

### 핸드 트래킹이 안 돼요

1. **OpenXR 설정 확인**
   - Project Settings > XR Plug-in Management > OpenXR
   - **Hand Tracking Subsystem** 체크

2. **Quest 3 설정 확인**
   - Quest 3 설정에서 핸드 트래킹 활성화

3. **XR Hands 패키지 확인**
   - Package Manager에서 `XR Hands` 패키지 설치됐는지 확인

## 성능 문제

### 화면이 끊겨요 (렉)

**해결 방법:**

1. **USB 3.0 케이블 사용**
   - USB 2.0은 대역폭이 부족해요

2. **품질 설정 낮추기**
   - Edit > Project Settings > Quality
   - 낮은 품질로 변경

3. **다른 프로그램 종료**
   - 크롬, 디스코드 등 종료
   - 특히 GPU 사용하는 프로그램

4. **Air Link → 유선으로 변경**
   - 무선보다 유선이 안정적이에요

5. **Refresh Rate 낮추기**
   - Quest 3 설정에서 90Hz → 72Hz로 변경

### VR이 너무 느려요

**최적화 방법:**

1. **오브젝트 수 줄이기**
   - 씬에 너무 많은 오브젝트가 있으면 느려져요

2. **그림자 끄기**
   - Light의 Shadow Type을 No Shadows로

3. **텍스처 크기 줄이기**
   - 너무 큰 텍스처는 성능에 영향

4. **Draw Call 줄이기**
   - 같은 Material을 사용하는 오브젝트들 합치기

## 기타 문제

### "Safe Mode" 창이 떠요

- **"Ignore"** 또는 **"No"** 선택하세요
- Safe Mode는 문제가 있을 때 쓰는 거예요

### Input System 재시작 요청

- **"Yes"** 클릭하세요
- Unity가 재시작되는 건 정상이에요

### 프로젝트가 안 열려요

1. **Unity 버전 확인**
   - 이 프로젝트는 Unity 2022.3.21f1이에요
   - 다른 버전이면 문제가 생길 수 있어요

2. **바이러스 백신 확인**
   - 가끔 바이러스 백신이 Unity를 막아요
   - 일시적으로 끄고 시도해보세요
