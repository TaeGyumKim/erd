# 사전 준비

VR 게임을 만들기 전에 준비해야 할 것들을 알아봅시다.

---

## 내 컴퓨터가 VR을 돌릴 수 있을까?

VR은 일반 게임보다 컴퓨터 사양이 더 필요해요. Quest Link(PC와 Quest 3를 연결하는 방식)를 사용하려면 아래 조건을 확인하세요.

### 최소 사양 (이 정도는 되어야 해요)

| 항목 | 필요 사양 |
|------|-----------|
| 운영체제 | Windows 10 또는 11 (64비트) |
| CPU (프로세서) | Intel i5-4590 또는 AMD Ryzen 5 1500X |
| RAM (메모리) | 8GB |
| 그래픽카드 | GTX 1060 또는 RX 580 |
| USB 포트 | USB 3.0 이상 |

### 권장 사양 (이 정도면 쾌적해요)

| 항목 | 권장 사양 |
|------|-----------|
| 운영체제 | Windows 11 (64비트) |
| CPU (프로세서) | Intel i7 또는 AMD Ryzen 7 이상 |
| RAM (메모리) | 16GB 이상 |
| 그래픽카드 | RTX 3060 또는 RX 6600 이상 |

### 내 컴퓨터 사양 확인하는 법

1. 키보드에서 `Windows 키 + R` 누르기
2. `dxdiag` 입력하고 엔터
3. "시스템" 탭에서 CPU, 메모리 확인
4. "디스플레이" 탭에서 그래픽카드 확인

## 준비물 체크리스트

VR 게임 개발에 필요한 것들이에요:

### 필수 준비물

- [ ] **Meta Quest 3 헤드셋**
  - Quest 2도 가능하지만, 이 프로젝트는 Quest 3 기준으로 만들어졌어요

- [ ] **USB-C 케이블** (Quest Link용)
  - USB 3.0 이상 지원 케이블이어야 해요
  - 3미터 이상 길이를 추천해요 (너무 짧으면 불편해요)
  - 또는 **5GHz Wi-Fi** 환경이 있으면 Air Link(무선)도 가능해요

- [ ] **Meta 계정**
  - Quest를 살 때 만들었을 거예요
  - 없다면 [meta.com](https://www.meta.com/)에서 만들 수 있어요

### 선택 준비물 (있으면 좋아요)

- [ ] **넓은 공간** - VR은 돌아다니며 플레이하니까요
- [ ] **편한 의자** - 오래 개발하면 힘들거든요

## Meta Quest 앱 설치하기 (PC용)

Quest Link를 사용하려면 PC에 Meta Quest 앱을 설치해야 해요.

### 설치 순서

**1단계: 다운로드 페이지 열기**

웹 브라우저를 열고 주소창에 입력하세요:
```
https://www.meta.com/quest/setup/
```

**2단계: 다운로드**

페이지에서 "Download Software" 또는 "소프트웨어 다운로드" 버튼을 클릭하세요.

**3단계: 설치하기**

1. 다운로드된 `MetaQuestSetup.exe` 파일을 더블클릭
2. 설치 마법사의 안내를 따라 "다음" 클릭
3. 설치 완료될 때까지 기다리기 (5~10분 정도)

**4단계: 로그인하기**

1. 설치가 끝나면 Meta Quest 앱이 자동으로 실행돼요
2. Meta 계정으로 로그인하세요
3. Quest 3에서 사용하는 것과 **같은 계정**으로 로그인해야 해요!
