# 프로젝트 열기

이 프로젝트를 Unity에서 열어봅시다!

---

## 프로젝트 열기

### 방법 1: Unity Hub에서 열기 (추천)

**1단계: Unity Hub 실행**

Unity Hub를 실행하세요.

**2단계: 프로젝트 추가**

1. 왼쪽 메뉴에서 **"Projects"** 클릭
2. 오른쪽 위 **"Open"** 버튼 클릭

**3단계: 폴더 선택**

1. 이 프로젝트가 있는 폴더를 찾아가세요
2. 폴더를 선택하고 **"Open"** 또는 **"폴더 선택"** 클릭

**4단계: 기다리기**

처음 열 때는 시간이 좀 걸려요:

- Unity가 필요한 패키지를 다운로드해요
- 프로젝트 설정을 가져와요
- **5분 ~ 15분** 정도 걸릴 수 있어요

> **중요!** 로딩 중에 Unity를 끄지 마세요! 프로젝트가 망가질 수 있어요.

---

## 처음 열 때 나올 수 있는 것들

### "Safe Mode" 물어볼 때

가끔 이런 창이 뜰 수 있어요:
```
"Enter Safe Mode?"
```

- **"Ignore"** 또는 **"No"** 를 선택하세요
- Safe Mode는 문제가 있을 때 쓰는 거예요

### 패키지 업그레이드 물어볼 때

```
"Do you want to upgrade packages?"
```

- **"No"** 를 선택하세요
- 버전이 바뀌면 호환이 안 될 수 있어요

### Input System 재시작 요청

```
"Active Input Handling을 변경하시겠습니까?"
"Unity를 재시작해야 합니다."
```

- **"Yes"** 를 클릭하세요
- Unity가 재시작되는 건 정상이에요

## Unity 화면 알아보기

프로젝트가 열리면 이런 화면이 보여요:

```
┌─────────────────────────────────────────────────────────────┐
│  메뉴바 (File, Edit, Assets, GameObject, ...)              │
├──────────────┬─────────────────────────┬───────────────────┤
│              │                         │                   │
│  Hierarchy   │     Scene / Game        │    Inspector      │
│  (씬 목록)    │     (게임 화면)          │   (속성 편집)      │
│              │                         │                   │
│              │                         │                   │
├──────────────┴─────────────────────────┴───────────────────┤
│                     Project (파일들)                        │
│              Console (로그/에러 메시지)                      │
└─────────────────────────────────────────────────────────────┘
```

### 각 창의 역할

| 창 이름 | 뭘 하는 곳인가요? |
|---------|-------------------|
| **Hierarchy** | 씬에 있는 모든 오브젝트 목록 |
| **Scene** | 게임 세계를 보고 편집하는 곳 |
| **Game** | 실제 게임 화면 미리보기 |
| **Inspector** | 선택한 오브젝트의 설정을 바꾸는 곳 |
| **Project** | 프로젝트 안의 모든 파일들 |
| **Console** | 오류 메시지나 디버그 로그 |

## 프로젝트 구조 확인하기

Project 창에서 **Assets** 폴더를 보면:

```
Assets/
├── Scripts/        ← 코드 파일들 (C#)
├── Scenes/         ← 게임 씬들
├── Materials/      ← 재질 (색상, 질감)
├── Prefabs/        ← 미리 만든 오브젝트들
├── Settings/       ← 프로젝트 설정
└── ...
```

지금은 구경만 해도 괜찮아요!

## 씬 열어보기

게임의 한 장면을 "씬(Scene)"이라고 해요.

**씬 여는 법:**

1. Project 창에서 **Assets > Scenes** 폴더 클릭
2. 씬 파일(`.unity`)을 더블클릭

아직 씬이 없다면 다음 단계에서 만들어볼 거예요!

## 문제가 있을 때

### "Version mismatch" 에러

```
"This project was created with a different version of Unity"
```

이 프로젝트는 Unity 2022.3.21f1로 만들어졌어요.
다른 버전으로 열면 이 메시지가 나올 수 있어요.

**해결법:** [Unity 설치](02-Unity-설치.md)에서 2022.3.21f1을 설치하세요.

### 패키지 에러 (빨간색 오류)

Console 창에 빨간색 오류가 잔뜩 뜬다면:

1. 잠시 기다려보세요 (패키지 설치 중일 수 있어요)
2. Unity를 껐다 다시 켜보세요
3. 그래도 안 되면 [문제해결](10-문제해결.md)을 확인하세요
