# 코드 리뷰

지정된 스크립트를 리뷰합니다.

## 사용법
`/review [파일경로]`

## 리뷰 항목

1. **Unity 베스트 프랙티스**
   - GetComponent 캐싱
   - Update에서 할당 최소화
   - 적절한 생명주기 메서드 사용

2. **VR 고려사항**
   - 프레임 드롭 원인
   - XR Interaction Toolkit 호환성
   - 멀미 유발 요소

3. **코드 품질**
   - 네이밍 컨벤션
   - 주석 및 문서화
   - 에러 처리

4. **보안**
   - 입력 검증
   - 널 체크

리뷰 대상: $ARGUMENTS
